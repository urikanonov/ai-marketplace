# assets

Canonical hand-edited sources for the commentable-html layer.

- `commentable-html.css` - layer stylesheet.
- `commentable-html.js` - runtime and `CMH_VERSION` source of truth.
- `template.shell.html` - shell with placeholders and commentable regions.

After edits, run the build from `dev/` and see `../README.md` for the full workflow.
