# Chart.js reference index

The former single Chart.js guide is split so an agent can load only the guidance it needs:

- [Chart.js embedding and tooltips](charts-embedding.md) - dependency choices, the four commenting-layer coexistence rules, minimal light-theme markup, tooltip options, tooltip verification, and embedding pitfalls.
- [Chart.js recipes, data hygiene, and themes](charts-recipes.md) - line, bar, horizontal ranking, dual-axis, logarithmic, stacked, value-label, data-cleanup, and dark-theme recipes.

If you are fixing a broken or blank chart, start with [Chart.js embedding and tooltips](charts-embedding.md). If the chart renders but needs a different visualization pattern or data treatment, use [Chart.js recipes, data hygiene, and themes](charts-recipes.md).
