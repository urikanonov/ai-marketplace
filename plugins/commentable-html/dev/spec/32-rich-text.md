## Rich-text comment notes

Review comment notes are stored as a plain-text markdown-ish SOURCE string and rendered to safe HTML
at display time (sidebar card, inline popover, print appendix). A composer formatting toolbar and
Ctrl/Cmd shortcuts make the markers discoverable, mimicking Office-style comments. Rendering is
presentation-only: the stored note is unchanged, so old notes stay compatible and Copy all preserves
the raw source.
